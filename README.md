# Data Structures

This repository contains **assignments, projects, and practice work** developed for the **Data Structures** course.  
It focuses on the implementation and analysis of fundamental data structures and algorithms, primarily using **C++**, with some complementary exercises in **Java**.

---

## Repository Overview

The goal of this repository is to provide a structured collection of academic work related to **data structures and algorithmic problem solving**, covering both theoretical concepts and practical implementations.

---

## Repository Structure

- `.qtcreator/`  
  Project files created using Qt Creator during early development stages.

- `.vscode/`  
  Visual Studio Code configuration files used for development and debugging.

- `CursoC++_Udemy/`  
  Supplementary C++ exercises focused on language fundamentals and control structures.
